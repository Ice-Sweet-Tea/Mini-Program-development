# 小程序项目集合
黑马学习视频(2021-12-17 09:30:11),地址：https://www.bilibili.com/video/BV1834y1676P
本仓库包含多个独立的小程序模块，用于不同场景的功能演示与开发实践。

## 模块列表

- **MiniProgram-1** 至 **MiniProgram-9**：9 个小程序基础示例
- **MiniProgram-副本**：某一个小程序的备份版本
- **local_life**：本地生活相关的小程序模块

## 提交记录

- 所有模块均为初始提交（Initial commit）
- 提交时间：约 9 分钟前
- 提交者：Ice-Sweet-Tea

## 使用说明

1. 克隆本仓库
2. 根据需要进入对应的小程序目录
3. 使用对应的小程序开发工具（如微信开发者工具）打开并运行

## 注意事项

- 各模块相互独立，互不影响
- 建议根据具体需求选择对应的模块进行开发和调试
